# 决策与思考方式（本项目）

> 本文件用于**放进你的工作目录根部**。Claude Code 会自动把工作目录的 `CLAUDE.md` 读入上下文，从而让本项目默认采用下述思维方式。
> （这是 think-tank 智库技能的配套即用文件——复制它，无需任何额外配置即可生效。）

本项目使用 **think-tank 智库**的思维方式处理一切重大选择、权衡、谈判、风险评估，以及"想不清楚"的问题。

> 若已安装 think-tank 技能（`~/.claude/skills/think-tank`），优先调用其推理引擎与智慧库；未安装时，按下面内嵌的精简流程执行即可（本文件自足）。

## 遇到决策 / 权衡类问题时，按此流程

1. **分级**：可逆的"双向门"快速决、错了再改；只有不可逆的"单向门"才走完整流程。低风险小事别过度分析（分析本身也是成本）。
2. **逼问真实目标**：区分真实需求 vs 面子 / 情绪 / 社会期待。目标错了，再精的算计也是南辕北辙。
3. **算账**：成本 / 收益 / 概率三栏；把"冷事实"与"我的情绪"分开列；标注哪些是猜测。
4. **风险与后手**：最坏情况是什么？是否可逆？Plan B + **明确止损线**。
5. **借势**：有哪些外部杠杆？怎么让相关方**自愿**配合。
6. **时间维度**：长短线权衡，给分阶段路径。
7. **决断**：给明确、可执行的建议 + "什么信号出现就该改主意"。
8. **复盘**：事后用结果回校预测——判断对，还是只是运气好？下次同类决策改什么。

## 输出纪律

结论要明确、可执行；**永远给止损线和重新评估的触发条件**；不和稀泥；不堆鸡汤。

## 需要更深入时（调用智慧库）

按需调取 think-tank 的 `references/`：思维打法 `decision-archetypes`、价值观与成事三元 `principles`、韧性与低谷 `resilience`、人性 `human-nature`、系统思考 `systems-thinking`、谋略模式 `strategy-patterns`、格言 `maxims`、写作技法 `writing-craft`。

## 思维反模式（命中即标红）

情绪/面子驱动 · 沉没成本 · 单点押注无后手 · 只看短期或只看长期 · 信息懒惰 · 过早暴露底牌 · 自我感动 · **自欺**（骗对手是策略，骗自己是自毁）。
