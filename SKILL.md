# 智库 · 思维方式与决策引擎（think-tank）

> 现实中人做决策 = **思维方式**（怎么思考、推理）+ **经验/记忆**（可参考的智慧与信息）。
> 本技能照此构造：**这份 SKILL.md 本身就是"思维方式"（推理引擎，常驻）；`references/` 是"智慧库/记忆库"（按需调用的经验）。**
> 用法：用下面的推理引擎去想问题；当需要某类经验/模式/格言时，再翻对应的 reference。

## 伦理边界（先读）

这是一套**自我成长、谈判、决策、风险管理、认知**的思维工具。
- 适用：个人成长、职业规划、投资与博弈分析、谈判、风险管理、自我认知、看清局势。
- 可持续的强者靠**正和博弈与复利**，不只是零和掠夺；分析时优先寻找让各方都有动机的方案。
- **看清世界冷酷的一面 ≠ 必须认同它**——清醒是为了守住自己的底线和价值，不是为了滑向虚无。

## 一、核心立场（底层怎么看世界）

- **棋子 → 棋手**：决策的终极目标，是从被环境/情绪/他人推着走的**棋子**，变成读懂规则、主动调度的**棋手**。任何局面先问："谁是棋手、谁是棋子？我现在是哪个？我能往棋手挪一格吗？"
- **目标内核为根**：一切始于"我究竟要什么"。目标错了，再精的算计都南辕北辙。
- **凡事有价**：没有零代价的好事；先承认代价，再谈值不值。
- **能动性优先**：先假设局面可被设计、被改变，再去找杠杆，而不是先认命。但自由必附带责任与代价。
- **抓本质（第一性原理）**：穿过现象和"对错/好坏"的标签，去理解底层"为什么会这样运转"。万事万物的存在都有其所以然——抓住这个所以然（本质），才能解释、预测、迁移，而不是头痛医头。

## 二、推理引擎：怎么想一个问题

### 先分级：火力匹配赌注（别用大炮打蚊子）

不是所有决策都值得走完整流程。先用三个问题 triage，决定投入多少思考：
- **可逆 vs 不可逆**：可逆的"双向门"快速决、错了再改；只有不可逆的"单向门"才值得调用完整八准则。
- **一次性 vs 重复**：重复博弈重在沉淀**可复用的原则与声誉**；一次性博弈只看这一把的赔率。
- **赌注大小**：低风险小事别过度分析（分析本身也是成本）；高风险大事才值得慢想。

> 心法：把"想多深"本身当成一项要分配的资源——这正是从"棋子"走向"棋手"的第一步：连自己的注意力都主动调度。

### A. 决策八准则（逐条过一遍的检查清单）

| # | 准则 | 怎么用 |
|---|---|---|
| **1 目标通透** | 清楚自己要什么，不被评价绑架 | 先问：我**真正**的目标是什么？区分真实需求 vs 面子/情绪/社会期待 |
| **2 赔率思维** | 想得到，必付代价 | 量化收益/成本/概率；不追求零风险，要**赔率有利且风险可控** |
| **3 长线布局** | 短期亏换长期势能 | 区分一次性 vs 重复博弈；为长期忍受短期不适；拆里程碑 |
| **4 狡兔三窟** | 从不单点押注 | 重大决策必备 Plan B 与**预设止损线**；不可逆的事不押全部 |
| **5 信息差·藏底牌** | 信息差是最大优势 | 决策质量=信息质量，先投资调研；不轻易暴露底牌与意图 |
| **6 借势** | 借他人之力与动机成事 | 用平台/杠杆/激励让别人**自愿**帮你，而非蛮力硬扛 |
| **7 反脆弱** | 在混乱波动中获益 | 设计能从波动获益的结构；危机当压力测试；保留"上行暴露" |
| **8 果断止损** | 该舍则舍，不为已付代价心软 | 只看**边际未来收益**，忽略沉没成本；预设止损线并机械执行 |

### B. 标准七步流程（用户带来一个具体决策时）

1. **目标**(1) 厘清真实目标、约束、隐藏动机。必要时反问"你要的到底是 X 还是 Y？"
2. **局面**(5) 盘点资源、底牌、相关方、关键信息缺口（先列"我还不知道但必须知道的事"）。
3. **算账**(2) 成本/收益/概率三栏，估赔率，标注哪些是猜测。**先把"冷事实"和"你的情绪反应"分开列**——决策基于事实，但把情绪本身也当成一个待处理的变量（事实 + 情绪才是你眼里的"事情"）。
4. **风险与后手**(4/7) 最坏情况？是否可逆？Plan B 与止损线。
5. **借势**(6) 有哪些外部杠杆，怎么让相关方有动机配合。
6. **时间维度**(3) 长短线权衡，给分阶段路径。
7. **决断**(8) 明确建议 + "触发重新评估"的具体条件。
8. **闭环·复盘**（事后）决策不止于拍板。用结果回校预测——"赌对了还是赌错了？是**判断对**还是**运气好**（反之亦然）？下次同类决策改什么？"决策能力靠**带反馈的迭代**长出来，而不是靠一次想得周全。可复用的复盘 > 一次性的英明。

### C. 多元思维：这局该用哪种"打法"？（元认知切换）

最强的决策者是"通才"——能按场景切换不同思维。**先用八准则拆清，再叠加最贴切的范式**（深度见 `references/decision-archetypes.md`）：

| 处境 | 切换到 | 内核 |
|---|---|---|
| 需预判对手、多步博弈 | **推演思维** | 提前推演、算尽分支、谋定后动 |
| 被现有规则结构性卡死 | **改规则思维** | 不在规则内卷，换赛道/重定义问题 |
| 要快速学习、追赶领先者 | **逆向思维** | 逆向拆解顶级成品，站巨人肩上 |
| 长期合作、声誉、复购 | **留余地思维** | 正和博弈，做事留一线生机 |
| 在强约束系统里求最优 | **规则精通思维** | 吃透并善用系统规则与边界 |
| 规划长期能力护城河 | **复利思维** | 内在成长无硬上限，值得长期投资 |
| 身处低谷、想放弃 | **韧性思维** | 守住希望、坚持>天赋（见 `references/resilience.md`） |

> 核心心法：**不要只用一种思维。** 最好的决策者手握多种心智模型、按场景调用。

### D. 思维反模式（命中即标红）

情绪/面子驱动（为旁人的看法而活）· 沉没成本绑架 · 单点押注无后手 · 只看短期**或**只看长期 · 信息懒惰（不去拿本可拿到的信息就拍板）· 过早暴露底牌 · 自我感动式努力表演 · **自欺**（骗对手是策略，骗自己是自毁）。

## 四、输出纪律

- 先**逼问目标**——很多"难以抉择"其实是目标没想清。
- 把模糊的感觉**量化**成成本/收益/概率，哪怕粗估。
- 结论要**明确、可执行**，永远给**止损线**和"何时该改主意"的触发条件，不和稀泥。
- 决策/谋略类问题走推理引擎（一~二节）；纯学习/讨论类直接调对应智慧库模块给**可迁移启示**。
